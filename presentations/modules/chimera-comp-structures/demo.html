In this session, we'll get into protein structure analysis:

  • looking at hydrogen bonds
  • contacts
  • sidechain conformations
  • mapping properties to structures
  • comparing different structures.
  1. Start the command line: Favorites→ Command Line
  2. Open 2gbp
    This is glucose-binding protein, dumbbell-shaped with glucose in the middle.
  3. Show all atoms: Presets→ Interactive 2
  4. Do File→Open and show all of the formats
    Talk about formats and cacheing
  5. Enable side view (Favorites→Side View)
  6. Show atom "baloons"
  7. Show various selection modes
  8. Look at binding site
    1. Select→Structure→ligand (or Select→Residue→BGC)
    2. Select→Zone→ [4.0 A, residues]
    3. Actions→Focus
    4. Actions→Atoms/Bonds→show only
    5. Actions→ Atoms/Bonds→stick
  9. Measure distances between two atoms (e.g.BGC O6 <-> ASN OD1).
  10. Use FindHBond (Tools→Structure Analysis→FindHBond
    [turn off relax criteria; linewidth 4; explain color wells, change color, OK]
  11. Turn off the distances and hbonds with ~dist and ~hbond in the command line
  12. Show help: Help→User's Guide

Now for some structure comparison. We have been looking at the closed, glucose-bound structure of the protein. Another structure of this protein has been solved in the open, unbound state.

  1. open 2fw0
  2. focus
  3. Presets→Interactive 1 (ribbons)
  4. Align using MatchMaker (show Tools→Structure Comparison→MatchMaker, but use mm #0 #1 in the command line
  5. focus
  6. Bring up Morph: Tools→Structure Comparison→Morph Conformations
    Make sure to click "Hide Conformations"
  7. Show how to selectively show some atoms (e.g. show aromatic)
    (if asked, could show ligand with: modeldisp #0; ~ribb #0; show ligand)
  8. File→Close Session

Finally, I'd like to talk about attributes, properties of atoms and residues.

  1. open 1hxx.pdb
  2. Presets→ Interactive 1 (ribbons)
  3. ~disp
    This is a transmembrane protein, a porin.
  4. Tools→Structure Analysis→Render by Attribute
  5. [res avg bfactor, Colors; move red to middle, change white to yellow, Apply]
  6. [residue kdHydrophobicity, Worms, Apply]
    To show regular ribbons later, it is necessary to render again with Worm style = non-worm [OK]

    7. You can create your own custom attributes by reading in text files in a simple format; the User's Guide includes several examples.

    New attributes are also created by various operations in Chimera. For example, showing a surface creates surface area attributes, and opening an associated sequence alignment assigns conservation values to residues. You'll learn more about that in the next session.
