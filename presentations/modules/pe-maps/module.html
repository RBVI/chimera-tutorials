Visualization Challenge

  • Typical visualization for E-MAPs:
    • Clustered heatmap
  • Issues:
    • No visual tie to structure
    • Difficult to explore data

Exploratory Visualization

  • Utilize network representation that combines
    • Clustered genes
      • Organized to match the hierarchical cluster
    • Residue Interaction Network (RIN)
    • Edges represent interactions between them
  • Integrate with 3D structural view
  • Provide interactive tools to explore data:
    • Search interface for gene or residue
    • Filter by the number of interactions
    • Brushing and linking between views
