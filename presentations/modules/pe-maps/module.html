pE-MAPS

  • Typical visualization for E-MAPs:
    • Clustered heatmap
  • Issues:
    • No visual tie between genes and structure
    • Difficult to explore data
  • Previous approach:
    • Python script that generated image for each complex of interest

pE-MAPS

  • Chewing gum and bailing wire approach
    • Utilize tools already familiar to researchers
      • Cytoscape -- network visualization and analysis
      • UCSF ChimeraX -- structural visualization
    • Combine with existing Cytoscape apps:
      • clusterMaker2: hierarchical cluster visualization
      • RINalyzer: 2D network representation of structure
      • structureVizX: link between Cytoscape and UCSF ChimeraX
      • setsApp: ability to select "sets" of nodes
    • Write new app (StEMAP) to provide "glue" and additional functionality (filtering, filtered heatmap)
    • Supports use of additional tools:
      • UCSF ChimeraX point mutation modeling
      • Cytoscape enrichment analysis and access to PPI databases (e.g. STRING)

Exploratory Visualization

  • Utilize network representation that combines
    • Clustered genes
      • Organized to match the hierarchical cluster
    • Residue Interaction Network (RIN)
    • Edges represent interactions between them
  • Integrate with 3D structural view
  • Include capability to optionally view the original heatmap
  • Provide interactive tools to explore data:
    • Filter by the number of interactions or complex
    • Search interface for gene or residue
    • Brushing and linking between views
    • Overview+detail local heatmap/network overview
