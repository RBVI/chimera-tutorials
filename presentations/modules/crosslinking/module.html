Chemical Cross-linking + Mass Spectrometry

  • What is it?
    1. Protein cross-linking reaction where there are a number of different linker types available including the ones with stable isotopes to enable detection and quantification
    2. Isolating/separating the cross-linked proteins from non-cross-linked to reduce the false positive rate
    3. Digestion of proteins into peptides by enzymes or chemical reagents
    4. Cross-linked peptide enrichment by physio-chemical properties
    5. MS method optimization for cross-linked peptide detection
    6. Data processing to map cross-linked sites

B.Q. Tran et al. / Biochimica et Biophysica Acta 1864 (2016)

Chemical Cross-linking + Mass Spectrometry

  • What is it used for?
    • Determine binding poses of complexes.
    • Stabilize (possibly transient or semi-stable) protein tertiary and quaternary structure for analysis.
    • Capture and identify unknown protein interactors or interaction domains.
    • Conjugate an enzyme or tag to an antibody or other purified protein.
    • Immobilize antibodies or other proteins for assays or affinity-purification.
    • Attach peptides to larger "carrier" proteins to facilitate handling/storage.

Installation
  1. Download XLinkAnalyzer and unzip it somewhere
    • You can also download the documentation, if desired
  2. Start Chimera
  3. Go to Menu Favorites→Preferences...→Category: Tools
  4. Add... button, which is on the bottom of the window, and browse for directory of Xlink Analyzer.
    • So if your unzipped Xlink Analyzer has path: /user/Desktop/XlinkAnalyzer/, add /user/Desktop/XlinkAnalyzer. A line similar to “XlinkAnalyzer - /user/Desktop/” should appear in Locations section of the Tools window
  5. Click Save
  6. Close. This configuration will persist and you don't need to repeat it. NOTE: Do not remove the XlinkAnalyzer directory

Chemical Cross-linking in Chimera

  1. Download bovineRNAPolII.zip
    • This is a data set from Structure of transcribing mammalian RNA polymerase II , an effort that combined use of a 3.4Å Cryo-EM map with chemical cross-linking to explore the structure of bovine RNA Polymerase II
  2. Unzip the file in a convenient directory
  3. Open the PDB structure (5FLM)
  4. Load the project ("bovineRNAPolII.json") into XLinkAnalyzer
  5. Explore the structure and hypothesize about the violated constraints
