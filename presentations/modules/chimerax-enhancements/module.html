ChimeraX Key Enhancements

  • Performance:
    • High-performance manipulation and rendering of large numbers of atoms
    • Fast mmCIF reader
    • Fast and robust solvent-excluded surfaces with clean patch boundaries
  • Visualization:
    • Interactive ambient occlusion lighting modes
    • Curved-cylinder “tube” helices integrated with cartoon style
    • Virtual reality camera mode (Oculus Rift, HTC Vive systems)

PDB: 5vms

ChimeraX Key Enhancements

  • User Interface:
    • Toolbars with icons to change styles, lighting , or mouse-button assignments
    • All-in-one window in which tool interfaces are panels
    • Interactive 2D plots and diagrams
    • Built-in HTML browser with ChimeraX command links

ChimeraX Key Enhancements

  • Extensibility:
    • Documented APIs with strict semantic versioning
    • Well-documented command interface
    • REST interface to all commands
    • Built-in browser can execute ChimeraX commands via custom links
    • Toolshed (web repository) of ChimeraX tool bundles
