High-Quality Images

UCSF Chimera has a number of features for producing high-quality images suitable for publication. Useful features include:

  • transparency
  • clipping
  • shadows
  • silhouettes
  • supersampling
  • special coloring
  • 3D labels
  • 2D labels
  • geometric objects
  • background coloring
